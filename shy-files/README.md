1. Dropout as a Bayesian Approximation: Representing Model Uncertainty in Deep Learning

基于 Dropout 的 Monte Carlo (MC) 采样是一种简单有效的贝叶斯深度学习方法，用于量化模型预测的不确定性。它通过在测试阶段启用 Dropout，并对同一输入进行多次前向传播，生成预测分布，从而估计模型的不确定性。
